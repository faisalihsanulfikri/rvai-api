## 🎉 What's New

### Image Input Support
Users can now upload reference images along with text prompts. Generated images can be used as reference for subsequent generations, enabling iterative design workflows.

**Key Capabilities:**
- Upload reference images (JPG, PNG, GIF, WebP, etc.)
- Image preview thumbnail in form
- File validation (type & size: max 5MB)
- Generated images become reference input
- Form-based regeneration workflow
- Multiple iterative regenerations

## 🚀 User Workflows

### Generate with Image
1. Click image icon → Upload reference
2. Type prompt
3. Click "Create Image" or press Enter
4. Design generates using text + image reference

### Regenerate from Previous Output
1. Hover over generated image
2. Click refresh icon (Regenerate)
3. Form populates with previous prompt + generated image as reference
4. Modify prompt (optional)
5. Submit to generate variation
