## 🎯 Next Steps

**Ready to start?**

```bash
# 1. Install
npm install

# 2. Run
npm run dev

# 3. Open browser
http://localhost:3000

# 4. Try the features:
# - Type a prompt and press Enter
# - Upload reference images with the image icon
# - Hover over generated images
# - Click refresh icon to regenerate with previous output as reference
# - Click download icon to save images
```
